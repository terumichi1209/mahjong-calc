# 麻雀点数計算アプリ

Expo (React Native) で作った門前手専用の麻雀点数計算アプリ。

## 機能

### 手牌入力
- 牌画像をタップして手牌を選択（最大13枚）
- 13枚揃ったあとに選んだ牌が自動的に和了牌になる
- 手牌・和了牌をタップすると取り消し
- 暗カン対応（最大4つ）

### 自動計算
- 手牌14枚（暗カンがある場合はそれ以上）が揃うと自動で役・点数を計算
- 符計算・翻数計算・満貫以上のラベル表示

### 設定
| 項目 | 内容 |
|------|------|
| 場風 | 東・南・西・北 |
| 自風 | 東・南・西・北 |
| 和了 | ロン・ツモ |
| 役 | リーチ ON/OFF |
| カン | 暗カン宣言モード |
